# Project

## 📄 ATS Resume Optimizer & Resume Generator

Today I built and deployed an AI-powered ATS Resume Builder designed to help candidates create professional resumes that pass Applicant Tracking Systems (ATS) and attract recruiters.

The application analyzes resume content, identifies improvement opportunities, and generates ATS-friendly resumes aligned with modern hiring practices.

---

# Live Demo

🌐 https://ats-optimizer-blue.vercel.app/

---

# Problem Statement

Many qualified candidates struggle to get interview calls because their resumes:

- Are not ATS-friendly
- Lack relevant keywords
- Have poor formatting
- Do not align with job descriptions
- Fail automated screening systems

This project aims to solve these challenges using AI-powered resume analysis and optimization.
