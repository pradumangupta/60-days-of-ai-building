# Biggest Insights

### 1. Model Selection Matters

Different models are optimized for different tasks.

### 2. More Reasoning Is Not Always Better

Using maximum effort for every task can reduce productivity.

### 3. Context Still Matters

Even the best model performs better with proper context.

### 4. Sonnet Is the Best Daily Driver

For coding, learning, documentation, and AI projects, Sonnet provides the best balance of speed and capability.

---

### Effort
