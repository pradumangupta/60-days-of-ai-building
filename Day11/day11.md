# Problem Statement

Many qualified candidates get rejected before a recruiter even sees their resume.

The primary reasons include:

- Poor ATS compatibility
- Missing keywords
- Weak formatting
- Lack of role-specific optimization

This project aims to solve these challenges using AI.
